# Parcel Bundler Advantages

- Dev Builds
- Local Server
- HMR
- File Watching Algo - for any change made in files are reflected on the local env
- Caching - for faster builds (.parcel-cache folder)
- Image Optimization
- Minification
- Bundling
- Compress
- Code Splitting
- Differential Bundling (support older browsers)
- Diagnostics (provide better error descriptions with suggestion)
- Tree Shaking
- Different build for dev (npx parcel index.html) or prod (npx parcel build index.html)
